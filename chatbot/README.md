# Smart Chatbot

## About

Smart Chatbot is a simple Python project that can have basic conversations with users. It can answer common questions, tell jokes, display the current date and time, and store chat history during a session.

## Features

* Interactive chat interface
* Random responses
* Date and time information
* Jokes and basic conversation
* View and clear chat history
* Menu-driven program

## How to Run

```bash
python chatbot.py
```

## Menu Options

1. Start Chat
2. View Chat History
3. Clear Chat History
4. Exit

Type **bye** during a conversation to end the chat.
